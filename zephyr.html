Zephyr Video Game
Console Interface

The main objective behind this project was to either redesign an existing digital interface or create a new one from scratch in Figma. The route I chose to go down for this project was to design my own video game console interface. This prototype gives the general idea of the layout and functionality of what my vision of a console interface would look like. This interface has gone through various changes since I completed UX 1 to further refine its functionality and design. This is likely one of those projects that I will continue to improve over time; however, it is in a pretty good state as it is right now.
